The foundational concepts behind Gearbox Agentic.

- [**The Agent Loop**](/developers/ga-agent-loop) — The 6-step cycle: Discover → Analyze → Propose → Preview → Execute → Monitor. Event-driven review triggers adapt pace to urgency.

- [**Architecture & Tools**](/developers/ga-architecture) — How agents and frontends share the same SDK. The Tool Visualization Map: one SDK method → one MCP tool → one UI component.

- [**MCP Server**](/developers/ga-mcp) — 16 tools exposed as MCP tool calls for LLM agents, mapped directly to SDK methods.
